  • CURAITOR: An AI curator using OpenAI's API (image color correction, music generation)

  • GITHUB

  • 2023 12 17

Implementation

fix_image.py

  • Using GPT-4, adjust the RGB values of the input image.
  • reload information on suitable color schemes for eight types of paintings into GPT.
  • GPT will classify the image and adjust the RGB values accordingly to make them more appropriate.

music_gerneration.py

  • We also take an input image from the user's exhibition and classify it to generate a keyword that represents the exhibition's mood.
  • There are a total of seven keywords, which we retrieve from GPT and use to influence music generation.
  • In the music generation code, we have classical music files, each representing a specific mood. Using these files, new music is generated.
