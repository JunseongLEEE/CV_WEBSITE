YAI_toyproject

Description

  • YAI Toyproject : Translate Human emotion From a dog's perspective

  • GITHUB

  • 2024 08 31

We use EfficientNetV2, which improves training speed, model size, and memory efficiency from the original EfficientNet by utilizing Compound Scaling, Fused MBConv, and Progressive Learning.

Dataset

  • Use dog image (from Kaggle)
  • 1865 images for each class, angry/alert/frown/relax/happy
  • 8:2 ratio for train/val set
  • Test for real human image (me)

Model training

  • Use Google Colab, TPU v2
  • Adjust output class : 1000 class(originally) -> 5 class
  • Optimizer: RMS Prop, Scheduler: Cosine Annealing LR
  • Result = train accuracy: 86.79, validation accuracy: 72.91
